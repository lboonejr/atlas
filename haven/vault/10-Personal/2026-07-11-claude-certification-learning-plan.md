# Claude certification by end of August — curriculum + learning plan

Goal: Lemar wants a Claude/Anthropic certification in hand by end of August 2026.

## What exists (checked 2026-07-11, not assumed)
Anthropic's first official technical cert, **Claude Certified Architect (CCA) —
Foundations**, launched 2026-03-12. It is a proctored 60-question, closed-book exam
covering agentic architecture, MCP integration, Claude Code workflows, prompt
engineering, and context management. **Access is currently exclusive to Claude Partner
Network member organizations** — the first 5,000 partner-company employees get free
early access. Separately, Anthropic made 19 of its own courses free with real
certificates (Claude Code, the API, MCP, agent skills, subagents).

**Open question for Lemar:** the CCA exam itself may require Cuzzie's (or another org)
to be a Claude Partner Network member — worth confirming eligibility before building a
plan around sitting that exact exam, vs. treating the 19 free courses' certificates as
the near-term goal instead (achievable solo, no partner-org gate).

## Draft learning plan (8 weeks, Jul 14 → Aug 31)
1. Weeks 1-2: Claude Code fundamentals + Anthropic API basics courses.
2. Weeks 3-4: Model Context Protocol (MCP) + agent skills courses.
3. Weeks 5-6: Subagents, context management, prompt engineering courses.
4. Week 7: Confirm Claude Partner Network eligibility for Cuzzie's; if eligible, register
   for the CCA — Foundations exam.
5. Week 8: Review + sit the exam (if eligible) or complete remaining free-course
   certificates as the milestone.
