**Kind:** vendor
**Serves:** cuzzies, station

## Summary
Dutchie is the e-commerce / menu platform. Like Harvest Moon, it is a
**cross-domain** vendor — it serves both Cuzzie's and The Station off one shared
catalog — so it lives in `Entities` and is never filed under a single store.
